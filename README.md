# FlowMotion Mobile Core
This package contains a selection of utility APIs and method for mobile app development. To download this package, run the following command in your project. 
```npm i @flow-motion-io/mobile-core```
If you have not set up authentication for the FlowMotion NPM registry, please read the following guide on how to do so [Flowmotion Sharepoint](https://flowmotionltd.sharepoint.com/:w:/r/sites/FlowMotionHub/_layouts/15/Doc.aspx?sourcedoc=%7BA4DE2816-BAC0-4340-A251-51FFE77F3FBE%7D&file=Setup%20FlowMotion%20NPMRC.docx&action=default&mobileredirect=true).
